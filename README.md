### 1. 3D 객체의 렌더링과 표시

| 객체 | 설명 |
| --- | --- |
| Plane | 배경으로 사용할 2차원 사각형|
| Cube | 빨간색으로 렌더링되는 3차원 정육면체 |
| Sphere | 파란색으로 렌더링되는 3차원 구체 |
| Camera | 결과물을 어떻게 보여질 것인지 결정 |
| Axes | x, y, z 축, 객체가 3차원 공간에서 렌더링되는 위치를 확인하는 유용한 디버깅 툴 |
